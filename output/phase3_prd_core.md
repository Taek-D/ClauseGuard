## 1. 프로젝트 개요

### 미션

> **계약서 리뷰를 법률 전문가의 전유물에서 모든 비즈니스의 기본 역량으로 전환한다.**

### 비전

2028년까지 SMB/중견기업이 체결하는 B2B 계약의 **30% 이상**이 AI 사전 리뷰를 거치는 세상을 만든다. ClauseGuard는 그 표준 도구가 된다.

### 핵심 가치 제안 (Core Value Proposition)

| 차원 | 현재 상태 (As-Is) | ClauseGuard (To-Be) |
|------|-------------------|---------------------|
| **비용** | 변호사 검토 건당 $500~$5,000 | 월 $99~$499 무제한 리뷰 |
| **속도** | 계약서 1건 리뷰에 3~7일 | **5분 이내** 자동 분석 완료 |
| **접근성** | 법무팀 또는 외부 로펌 의존 | 비법률 직군도 즉시 사용 가능 |
| **일관성** | 검토자마다 기준 상이 | 산업별 표준 리스크 프레임워크 기반 |
| **커버리지** | 핵심 계약만 선별 검토 | **모든 계약** 전수 리뷰 가능 |

### 원라이너

**"업로드하면 5분 안에, 변호사 없이도 계약 리스크를 잡아내는 AI 계약서 리뷰어"**

### 핵심 차별점

1. **가격 공백 공략** — 엔터프라이즈 CLM($15K+/년)과 무료 AI 도구 사이의 $100~$999/월 구간에 최적화
2. **SMB 맞춤 UX** — 법률 용어를 비즈니스 언어로 번역, 법무팀 없는 조직도 즉시 활용
3. **산업별 리스크 템플릿** — SaaS, 제조, 유통, 부동산 등 산업 특화 조항 분석
4. **한국어 + 영어 네이티브** — 국내 기업의 국문/영문 혼합 계약서 동시 지원

## 2. 타겟 사용자 & 유저 저니

### 2.1 타겟 사용자 페르소나

#### Persona A: 스타트업 COO "민지" (Primary)

| 항목 | 상세 |
|------|------|
| **역할** | 시리즈 A 스타트업 COO / 경영지원 총괄 |
| **회사 규모** | 20~50명 |
| **법무 인력** | 없음 (외부 로펌 스팟 의뢰) |
| **월 계약 건수** | 5~15건 (SaaS 구독, 파트너십, NDA, 용역) |
| **핵심 Pain** | 계약서 리뷰 요청 시 로펌 응답까지 3~5일 소요, 건당 50~200만원 |
| **핵심 Goal** | 명백한 리스크 조항을 사전에 걸러내고, 정말 필요한 건만 변호사에게 의뢰 |
| **기술 숙련도** | Google Workspace / Notion 능숙, 법률 지식은 기초 수준 |

#### Persona B: 중견기업 법무 담당자 "승환" (Secondary)

| 항목 | 상세 |
|------|------|
| **역할** | 매출 500억 중견기업 법무팀 사원 (팀 3명) |
| **회사 규모** | 200~500명 |
| **월 계약 건수** | 30~80건 |
| **핵심 Pain** | 반복적인 NDA/기본계약서 검토에 시간 소모, 핵심 거래 검토에 집중 불가 |
| **핵심 Goal** | 정형 계약은 AI로 자동 처리하고, 비정형/고위험 건에 리소스 집중 |
| **기술 숙련도** | CLM 도구 경험 있음, 법률 전문가 |

#### Persona C: 프리랜서/소상공인 "지수" (Tertiary)

| 항목 | 상세 |
|------|------|
| **역할** | IT 프리랜서 또는 1인 사업자 |
| **월 계약 건수** | 1~5건 |
| **핵심 Pain** | 계약서를 "읽긴 읽지만" 리스크 조항을 식별할 역량 부족 |
| **핵심 Goal** | 불리한 조항을 사전에 인지하고 협상 포인트 확보 |

## 3. 유저 스토리 (10개)

### P0 — Must Ship (MVP 필수)

#### US-01: 계약서 업로드 및 AI 분석

| 항목 | 내용 |
|------|------|
| **As a** | 스타트업 COO (민지) |
| **I want to** | PDF/DOCX 계약서를 업로드하면 AI가 자동으로 분석해주길 |
| **So that** | 변호사 없이도 5분 내에 계약서의 핵심 리스크를 파악할 수 있다 |

**Acceptance Criteria:**
- [ ] PDF, DOCX, HWP 형식 지원 (최대 50페이지, 20MB)
- [ ] 업로드 후 5분 이내에 분석 완료 (95th percentile)
- [ ] 분석 진행률 실시간 표시 (업로드 → OCR → 조항 파싱 → 리스크 분석 → 완료)
- [ ] 한국어, 영어 계약서 모두 지원
- [ ] 분석 실패 시 사유와 함께 에러 메시지 표시

---

#### US-02: 리스크 리포트 조회

| 항목 | 내용 |
|------|------|
| **As a** | 스타트업 COO (민지) |
| **I want to** | 분석 결과를 리스크 수준별로 정리된 리포트로 보고 싶다 |
| **So that** | 어떤 조항이 위험한지, 왜 위험한지 비법률 용어로 이해할 수 있다 |

**Acceptance Criteria:**
- [ ] 리스크 등급 3단계 표시: 🔴 High / 🟡 Medium / 🟢 Low
- [ ] 각 리스크 항목에: 원문 조항 하이라이트 + 평문 설명 + 위험 사유
- [ ] 전체 리스크 스코어 (0~100) 대시보드 상단에 표시
- [ ] 리스크 항목 클릭 시 원문 해당 위치로 스크롤 이동
- [ ] 리포트 PDF 다운로드 가능

---

#### US-03: 수정 제안 (Redline Suggestion)

| 항목 | 내용 |
|------|------|
| **As a** | 스타트업 COO (민지) |
| **I want to** | 위험 조항에 대해 AI가 대체 문구를 제안해주길 |
| **So that** | 상대방에게 바로 수정 요청을 보낼 수 있다 |

**Acceptance Criteria:**
- [ ] High/Medium 리스크 조항 각각에 1~2개의 대안 문구 제시
- [ ] 원문 vs 수정안 diff 형태로 시각적 비교 가능
- [ ] 수정 제안 선택 시 원문에 레드라인 마크업 자동 적용
- [ ] 레드라인 적용된 DOCX 파일 내보내기 지원
- [ ] 수정 제안에 "이 수정이 필요한 이유" 한 줄 설명 포함

- [ ] 유형별 맞춤 리스크 체크리스트 자동 적용
